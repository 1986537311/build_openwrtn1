## 简介

本项目用于编译打包 `斐讯N1` 使用的 openwrt/immortalwrt 固件。

## 说明

- Releases 中固件根据源码及编译时间不同，其包含的插件也略有差异。
- 如要 frok 使用务必添加 `GH_TOKEN` 至 secret，想要编译完成后收到 Telegram 通知，需要在 secret 中添加 `TELEGRAM_TO` (用户id)，`TELEGRAM_TOKEN` (tg token)。
